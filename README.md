# 📜 AI Spiritual Assistant

An AI-powered web application that provides contextual interpretations of scriptures from the **Bhagavad Gita**, **Quran**, and **Holy Bible** using Natural Language Processing (NLP) and Generative AI.

---

## 🚀 Features

- 📖 Supports Bhagavad Gita, Quran, and Holy Bible
- 🤖 AI-powered interpretations using Google Gemini
- 😊 Sentiment Analysis using VADER
- 🔍 Semantic Verse Retrieval using Sentence-BERT
- 📚 JSON-based Knowledge Base
- ⚖️ Multiple Interpretation Modes
  - Simple Explanation
  - Spiritual Insight
  - Historical Context
- 🔄 Compare Interpretations
- 🎨 Historic-themed Streamlit UI

---

## ⚙️ Installation

Clone the repository

```bash
git clone https://github.com/AliAbbas160/AI-Spiritual-Assistant.git
```

Install dependencies

```bash
pip install -r requirements.txt
```

Run the application

```bash
streamlit run app.py
```
